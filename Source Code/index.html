Technical Architecture Summary:

This engine was engineered to solve the scalability bottlenecks of traditional grid-based games. Unlike standard implementations that iterate through the entire board every frame, this engine utilizes QuadTree Spatial Indexing to achieve $O(\log N)$ viewport culling, allowing for boards with up to 1,000,000 cells at a constant 60 FPS.

  • CSP Solver: A backtracking Constraint Satisfaction Problem solver ensures every board generated is logically solvable without guessing.
  • Bit-Packed Data: State is managed using Uint32Array bitmasks for cache-friendly memory access and minimal GC overhead.
  • Canvas Pipeline: High-performance rendering via raw Canvas API with custom dirty-rect tracking to minimize redraw cycles.

How To Play

The goal of the game is to uncover all the squares that do not contain mines.

  • Left-Click: Reveal a square.
  • Right-Click: Plant a flag on a suspected mine.
  • Middle-Click: Pan the camera for large maps.
  • Scroll: Zoom in and out.

If you reveal a mine, the game is over!
