About Minesweeper

Minesweeper

A high-performance Minesweeper engine.

  • QuadTree spatial indexing for massive boards
  • Constraint Satisfaction solver for no-guess boards
  • BFS flood-fill with bitmask optimization
  • Bit-packed TypedArrays for memory efficiency
  • Seeded PRNG for reproducible, shareable games
