About Minesweeper

Minesweeper Engine

This project is a technical demonstration of high-performance board generation and spatial management.

  • Spatial Indexing: QuadTree-based partitioning for O(log N) cell retrieval. Handles 1,000,000+ cells at 60 FPS viewport culling.
  • Logic Solver: Deterministic backtracking CSP (Constraint Satisfaction Problem) solver. Validates board solvability with zero-guess guarantees.
  • Data Structures: Bit-packed Uint32Array bitmasks for raw state tracking. Optimized for CPU cache locality and minimal GC pressure.
  • Rendering: Stateless Canvas API pipeline with manual dirty-rectangle optimization.

Author: Amey Thakur

Source: https://github.com/Amey-Thakur/MINESWEEPER

How To Play

The goal of the game is to uncover all the squares that do not contain mines.

  • Left-Click: Reveal a square.
  • Right-Click: Plant a flag on a suspected mine.
  • Middle-Click: Pan the camera for large maps.
  • Scroll: Zoom in and out.

If you reveal a mine, the game is over!
